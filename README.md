# Prepora

## Setup

Before running the application, make sure to configure the environment variables:
1. In the `backend` folder, copy `.env.example` to `.env` and fill in your API keys (especially `GROQ_API_KEY` for Groq LLM and Whisper).
2. In the `frontend` folder, copy `.env.example` to `.env.local` to configure the `NEXT_PUBLIC_API_URL`.
